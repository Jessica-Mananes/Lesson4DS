CARDINALITY

The cardinality of a set is a measure of a set's size, which refers to the number of elements in the set.

B = { a, e, i, o, u}

For instance, the set above has a cardinality of 5, |B| = 5 ,for there are five elements that are inside the set.

CARDINALITY

Although cardinality refers to the number of elements in a set, it does not count separately the elements of the set that have the same value.

C = { 1, 3, 5, 5, 3, 1}

Set C still has a cardinality of 3, |D| = 3 despite having 6 elements since cardinality does not account for individual values of repeated numbers.
